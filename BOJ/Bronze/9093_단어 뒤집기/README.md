# BOJ 9093 단어 뒤집기

문제 링크  
https://www.acmicpc.net/problem/9093

## 문제 설명
문장이 주어지면 각 단어를 뒤집어서 출력하는 문제이다.  
단어의 순서는 유지하고, 각 단어 내부의 문자 순서만 뒤집는다.

예시

입력
I am happy

출력
I ma yppah


## 알고리즘
문자열 처리 (String)

## 풀이 아이디어

1. 테스트 케이스 개수를 입력받는다.
2. 문장을 공백 기준으로 분리한다.
3. 각 단어를 뒤집는다.
4. 뒤집은 단어를 공백으로 이어서 출력한다.

Python에서는 문자열 슬라이싱 `[::-1]`을 사용하면 쉽게 뒤집을 수 있다.

## 시간복잡도

O(N)
