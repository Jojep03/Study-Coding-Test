# BOJ 1157 - 단어 공부

## 🔗 문제 링크

https://www.acmicpc.net/problem/1157

---

## 📌 문제 설명

알파벳 대소문자로 이루어진 단어가 주어진다.

이 단어에서 **가장 많이 사용된 알파벳**을 출력하는 문제이다.

* 대소문자는 구분하지 않는다.
* 가장 많이 사용된 알파벳이 여러 개라면 `?`를 출력한다.

---

## 🧠 알고리즘

* 문자열 처리
* 구현
* 해시 (딕셔너리)

---

## 💡 풀이 아이디어

1. 입력 문자열을 모두 **대문자로 변환**하여 처리한다.
2. `A ~ Z` 각각의 등장 횟수를 딕셔너리로 저장한다.
3. 최댓값(`max_val`)을 구한다.
4. 해당 최댓값을 가진 알파벳을 모두 찾는다.
5. * 1개라면 → 해당 알파벳 출력
   * 2개 이상이라면 → `?` 출력

---

---

## ⚠️ 주의할 점

* `max(cnt)`는 **key 기준**으로 비교하므로 사용하면 안 된다.
* 반드시 `max(cnt.values())` 또는 `max(cnt, key=cnt.get)`를 사용해야 한다.
* 대소문자 구분을 없애기 위해 `upper()` 또는 `lower()`를 사용해야 한다.

---

## 🚀 한 줄 정리

👉 딕셔너리로 알파벳 개수를 세고, 최댓값을 가진 문자가 하나인지 여러 개인지 판별하면 된다.
