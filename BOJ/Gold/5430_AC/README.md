# 🥇 BOJ 5430 - AC

## 📌 문제 링크

https://www.acmicpc.net/problem/5430

---

## 📌 문제 설명

정수 배열과 함수 문자열 `p`가 주어집니다.

함수는 두 가지 연산으로 구성됩니다.

* `R` : 배열을 뒤집기
* `D` : 배열의 첫 번째 원소 삭제
  (배열이 비어있으면 `error` 출력)

함수를 순서대로 수행한 뒤 최종 배열을 출력하는 문제입니다.

---

## 💡 풀이 아이디어

### ❗ 핵심 포인트: 실제로 뒤집지 않는다

`R`이 나올 때마다 `reverse()`를 하면 시간초과가 발생합니다.
따라서 **배열을 실제로 뒤집지 않고 방향만 관리**합니다.

---

### 🔑 방법

* `rev`라는 boolean 변수를 사용

  * `False` → 정방향
  * `True` → 역방향

---

### ⚙️ 연산 처리

#### 1. R (뒤집기)

```python
rev ^= True
```

#### 2. D (삭제)

* 정방향이면 앞에서 제거
* 역방향이면 뒤에서 제거

```python
if rev:
    lst.pop()
else:
    lst.popleft()
```

---

### ❗ 예외 처리

* 배열이 비어있는데 `D`가 나오면

```python
error 출력
```


## ⏱️ 시간 복잡도

* 각 명령을 한 번씩만 처리 → **O(n)**
* `reverse()`는 최대 1번만 수행

👉 전체 시간 복잡도: **O(n)**


## 🧠 느낀 점

* `deque`의 양쪽 삭제 (`popleft`, `pop`)를 활용하는 것이 핵심
* **"연산을 실제로 수행하지 말고 상태만 관리"** 하는 최적화가 중요
* 구현 문제처럼 보이지만, 시간 최적화 아이디어가 핵심인 문제
