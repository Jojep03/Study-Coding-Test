# 🥇 BOJ 2110 - 공유기 설치

## 📌 문제 설명
집들의 좌표가 주어졌을 때, `C`개의 공유기를 설치하려고 합니다.  
단, 공유기 간의 **최소 거리**를 가능한 한 크게 하고 싶습니다.

즉, 공유기를 설치하는 모든 방법 중에서  
**가장 가까운 두 공유기 사이의 거리**가 최대가 되도록 배치해야 합니다.


## 💡 핵심 아이디어 (이분 탐색)

이 문제의 핵심은  
**"공유기 간 최소 거리"를 기준으로 이분 탐색**을 수행하는 것입니다.

- 공유기 간 최소 거리를 `mid`로 정하고  
- 이 간격 이상을 유지하면서 공유기를 `C`개 설치할 수 있는지 판단합니다.

판단 결과:
- `C`개 이상 설치 가능 → 거리를 더 늘려볼 수 있음
- `C`개 설치 불가능 → 거리를 줄여야 함


## ⚙️ 이분 탐색 범위 설정

- 가능한 최소 거리: `1`
- 가능한 최대 거리: `가장 오른쪽 집 - 가장 왼쪽 집`

```python
start = 1
end = homes[-1] - homes[0]
```

## 🧠 설치 방법
1. 첫 번째 집에는 무조건 공유기를 설치합니다.
2. 이후 집들을 순회하며, 이전 설치 위치와의 거리가 mid 이상이면 공유기를 설치합니다.
3. 설치한 공유기 수를 세어 C개 이상인지 확인합니다.


## 📌 예시

입력:
```
5 3
1
2
8
4
9
```

정렬된 집 위치: [1, 2, 4, 8, 9]

- 최소 거리 3 → 설치 가능 (1, 4, 8)
- 최소 거리 4 → 설치 불가능 (1, 8, 12)

따라서 **정답은 3**입니다.


## 🧾 시간 복잡도
* 집 개수: N
* 좌표 범위: 최대 10^9

정렬: ``O(N log N)``

이분 탐색: ``O(log(max_distance))``

각 탐색마다 선형 탐색: ``O(N)``

전체 시간 복잡도:
```
O(N log N + N log(max_distance))
```
