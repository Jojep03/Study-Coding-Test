# 🥇 BOJ 3079 - 입국심사

## 📌 문제 설명
N개의 입국심사대가 있고, 각 심사대마다 한 사람을 심사하는 데 걸리는 시간이 다릅니다.  
M명의 사람들이 줄을 서 있으며, 모든 사람이 심사를 받는 데 걸리는 **최소 시간**을 구하는 문제입니다.


## 💡 풀이 아이디어 (이분 탐색)

이 문제는 **시간을 기준으로 이분 탐색**을 수행합니다.

### ✔️ 핵심 관찰

어떤 시간 `T`가 주어졌을 때:

- 각 심사관이 처리할 수 있는 사람 수는  
```python
T // time[i]
```
- 전체 처리 가능한 사람 수는  
```python
sum(T // time[i])
```


### ✔️ 이분 탐색 조건

- `cnt >= M`  
→ 해당 시간 안에 M명 처리 가능  
→ 더 짧은 시간도 가능한지 탐색 (`end = mid - 1`)

- `cnt < M`  
→ 시간이 부족  
→ 더 긴 시간 필요 (`start = mid + 1`)


## ⚙️ 시간 복잡도

- 이분 탐색: `O(log(max_time * M))`
- 각 탐색마다 N번 반복

👉 전체: **O(N log M)**


## ❌ 시간 초과 코드 (잘못된 접근)

```python
for time in times:
    for i in range(1, m + 1):
        if time * i <= mid:
            cnt += 1
```

## 🔥 문제점
- 각 심사관마다 1 ~ m까지 모두 확인
- 시간 복잡도: O(N × M) → 매우 비효율적


## ✅ 핵심 정리
- 사람 수를 하나씩 늘려가며 확인 ❌
- 나눗셈 한 번으로 처리 가능 ⭕
```python
cnt += mid // time
```
