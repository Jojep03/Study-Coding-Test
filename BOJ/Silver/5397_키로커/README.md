# BOJ 5397 키로거

문제 링크  
https://www.acmicpc.net/problem/5397

## 문제 설명

키보드 입력 로그가 주어질 때 최종 문자열을 구하는 문제이다.

특수 문자

| 문자 | 의미 |
|----|----|
| < | 커서를 왼쪽으로 이동 |
| > | 커서를 오른쪽으로 이동 |
| - | 커서 왼쪽 문자 삭제 |

문자 입력 시 커서 위치에 문자가 추가된다.

## 알고리즘
스택 (Stack)

## 풀이 아이디어

문자열을 직접 수정하면 시간복잡도가 커지기 때문에  
**커서를 기준으로 두 개의 스택을 사용한다.**

left → 커서 왼쪽
right → 커서 오른쪽


처리 방식

- `<` : left → right 이동
- `>` : right → left 이동
- `-` : left pop
- 문자 : left append

마지막에 left + reversed(right)

로 문자열을 합친다.

## 시간복잡도

O(N)
