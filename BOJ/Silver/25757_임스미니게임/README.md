# 🥇 BOJ 25257 - 임스와 함께하는 미니게임

## 📌 문제 설명
임스와 함께하는 미니게임에서  
플레이어 수와 게임 종류가 주어질 때,  
최대 몇 번 게임을 진행할 수 있는지 구하는 문제입니다.

게임 종류는 다음과 같습니다.

- Y : 1명씩 플레이
- F : 2명씩 플레이
- O : 3명씩 플레이

※ 같은 사람이 여러 번 신청할 수 있지만, 중복은 제거해야 합니다.

---

## 💡 풀이 아이디어

1. 플레이어 이름을 입력받는다.
2. set을 사용하여 중복을 제거한다.
3. 게임 종류에 따라 나누어 계산한다.

- Y → 그대로 출력
- F → 2명씩 → len(players) // 2
- O → 3명씩 → len(players) // 3

---

## ⚙️ 핵심 포인트

- set()을 이용한 중복 제거
- 문자열 조건 분기
- 몫 연산(//) 활용


---

## ⏱️ 시간 복잡도

- 입력 처리: O(N)
- set 변환: O(N)

👉 전체 시간 복잡도: O(N)

---

## ✅ 한 줄 요약

👉 중복 제거 후, 게임 인원 수로 나누면 끝!
