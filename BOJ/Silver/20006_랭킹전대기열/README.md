# 📘 BOJ 20006 - 랭킹전 대기열

## 🔗 문제 링크
https://www.acmicpc.net/problem/20006

---

## 📝 문제 설명

온라인 게임에서 플레이어들이 순서대로 대기열에 들어온다.

각 플레이어는 **레벨과 닉네임**을 가지고 있으며,  
다음 규칙에 따라 방이 생성된다.

### 방 생성 및 입장 규칙

- 새로운 플레이어는 **기존 방에 입장할 수 있는지 먼저 확인**
- 조건을 만족하는 방이 있으면 **가장 먼저 생성된 방에 입장**
- 없다면 **새로운 방 생성**

---

## 📌 방 입장 조건

플레이어 `(레벨 l, 닉네임 n)`이 방에 들어가기 위해서는:

- 방의 인원 수 < `m`
- 방장의 레벨과 차이 ≤ 10

```python
abs(host_level - l) <= 10
```


## 알고리즘

- 구현
- 시뮬레이션
- 그리디


## 핵심 아이디어

플레이어를 한 명씩 보면서:
1. 기존 방들을 앞에서부터 탐색

2. 입장 가능한 방이 있으면 들어감

3. 없으면 새로운 방 생성

## 자료구조

```python
rooms = [
    [(level, name), (level, name)],
    [(level, name)]
]
```

## 시간 복잡도

- 최악: O(p²)

- 입력 제한 내에서 충분히 통과
