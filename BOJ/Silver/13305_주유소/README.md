# BOJ 13305 주유소

문제 링크  
https://www.acmicpc.net/problem/13305

---

## 문제 설명

N개의 도시가 일직선으로 연결되어 있다.  
각 도시 사이의 **도로 길이**와 각 도시의 **기름 가격**이 주어진다.

차는 **왼쪽 도시에서 오른쪽 도시로 이동**해야 하며  
기름은 **현재 도시에서만 구매할 수 있다.**

목표는 **마지막 도시까지 가는 최소 비용**을 구하는 것이다.

---

## 알고리즘

Greedy (그리디)

---

## 핵심 아이디어

미래의 가격을 예측해서 전략적으로 주유할 필요가 없다.

**지금까지 지나온 도시 중 가장 싼 가격으로 계속 주유하면 된다.**

이유

- 더 싼 도시가 나오면 그때부터 그 가격으로 주유
- 지금 도시가 가장 싸다면 최대한 이용

따라서

현재까지 최소 기름 가격 × 다음 도로 길이


를 계속 더해주면 된다.

---

## 풀이 과정

1. 첫 번째 도시의 가격을 최소 가격으로 설정
2. 도로를 하나씩 이동하면서
3. 현재 도시 가격과 최소 가격을 비교하여 갱신
4. 최소 가격으로 해당 도로 길이를 곱해 비용 추가


## 시간 복잡도

```python
O(N)
```

## 배운 점

### 이 문제는 그리디 알고리즘의 대표적인 예시이다.

### 핵심은
지금까지 등장한 최소 가격을 유지하면서 이동한다

미래의 정보를 미리 고려할 필요 없이
현재까지의 최소값만 유지하면 최적해가 보장된다.
