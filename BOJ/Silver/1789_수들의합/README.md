# BOJ 1789 - 수들의 합

## 문제

서로 다른 N개의 자연수의 합이 S일 때, 자연수의 개수를 최대로 하는 값을 구하는 문제이다.

## 아이디어

서로 다른 자연수의 개수를 최대화하려면 가장 작은 수부터 차례대로 사용하는 것이 유리하다.

즉,

* 1
* 1 + 2
* 1 + 2 + 3
* ...

처럼 누적합을 증가시키다가 처음으로 S를 초과하는 순간 직전까지 사용한 개수가 정답이 된다.

예를 들어 S = 200이라면,

* 1 + 2 + ... + 19 = 190
* 1 + 2 + ... + 20 = 210

이므로 20개는 사용할 수 없고, 최대 개수는 19개이다.

## 풀이 코드

```python
import sys

input = sys.stdin.readline

n = int(input())
res = 0
cnt = 0

while res <= n:
    cnt += 1
    res += cnt

print(cnt - 1)
```

## 코드 설명

```python
while res <= n:
```

현재 누적합이 N 이하인 동안 계속 자연수를 추가한다.

```python
cnt += 1
res += cnt
```

현재 사용할 자연수를 1 증가시키고 누적합에 더한다.

예를 들어 다음과 같이 진행된다.

| cnt | res |
| --- | --- |
| 1   | 1   |
| 2   | 3   |
| 3   | 6   |
| 4   | 10  |
| ... | ... |

누적합이 처음으로 N을 초과하면 반복문이 종료된다.

```python
print(cnt - 1)
```

반복문이 종료된 시점의 cnt는 이미 N을 초과하게 만든 수까지 포함하고 있으므로 1을 빼서 출력한다.

## 시간복잡도

자연수의 합은

1 + 2 + ... + k = k(k + 1) / 2

이므로 k는 대략 √N 크기이다.

따라서 시간복잡도는 **O(√N)** 이다.
