# 🔐 BOJ 4659 - 비밀번호 발음하기

## 📌 문제 설명
문자열이 주어질 때, 해당 문자열이 **발음하기 좋은 비밀번호인지** 판단하는 문제입니다.

입력은 여러 줄로 주어지며, `"end"`가 입력되면 종료됩니다.

비밀번호가 **acceptable** 하려면 다음 조건을 모두 만족해야 합니다.

1. 모음을 **하나 이상 포함**해야 한다.
2. 모음이 **3개 연속** 나오면 안 된다.
3. 자음이 **3개 연속** 나오면 안 된다.
4. 같은 글자가 **연속으로 2번** 오면 안 된다.  
   (단, `ee`, `oo`는 허용)

---

## 💡 풀이 아이디어

문자열 하나당 조건을 각각 따로 검사하면 됩니다.

### ✔️ 1. 모음 포함 여부
- 문자열을 순회하면서 모음이 하나라도 있으면 OK

### ✔️ 2. 3글자 연속 검사
- `i, i+1, i+2`를 확인해서
  - 모두 모음 → ❌
  - 모두 자음 → ❌

### ✔️ 3. 같은 글자 연속 검사
- `x[i] == x[i+1]`이면
  - `ee`, `oo`만 허용
  - 나머지는 ❌

---

## ⚙️ 핵심 포인트

- ### 모음 체크는 이렇게 구현
```python
if c in "aeiou":
```

---
- ### 3글자 연속 체크는 이렇게 구현
```python
if (세 글자 모두 모음) or (세 글자 모두 자음):
```
---

- ### 같은 글자 연속은 예외 처리 필수
```python
if x[i] == x[i+1] and x[i] not in "eo":
```

---

## ⏱️ 시간복잡도
- 문자열 길이를 L이라고 할 때
- 모음 검사: O(L)
- 3연속 검사: O(L)
- 중복 문자 검사: O(L)
### 👉 총 O(L)
