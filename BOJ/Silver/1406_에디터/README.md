# BOJ 1406 에디터

문제 링크  
https://www.acmicpc.net/problem/1406

## 문제 설명

문자열과 커서가 주어지고 여러 명령어를 수행한 뒤  
최종 문자열을 출력하는 문제이다.

명령어

| 명령어 | 의미 |
|------|------|
| L | 커서를 왼쪽으로 이동 |
| D | 커서를 오른쪽으로 이동 |
| B | 커서 왼쪽 문자 삭제 |
| P x | 문자 x 추가 |

## 알고리즘
스택 (Stack)

## 풀이 아이디어

문자열의 중간에서 삽입과 삭제가 많이 발생하기 때문에  
일반 리스트로 구현하면 시간초과가 발생한다.

따라서 커서를 기준으로 **두 개의 스택을 사용한다.**

left → 커서 왼쪽 문자열
right → 커서 오른쪽 문자열


명령 처리

- `L` : left → right
- `D` : right → left
- `B` : left pop
- `P x` : left append

마지막에는

left + reversed(right)

을 출력한다.

## 시간복잡도

O(N)
