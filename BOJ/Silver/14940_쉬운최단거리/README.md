# BOJ 14940 - 쉬운 최단거리

## 문제 링크
https://www.acmicpc.net/problem/14940

---

## 문제 설명

N x M 크기의 지도에서 다음과 같은 값이 주어진다.

- `0` : 갈 수 없는 땅
- `1` : 갈 수 있는 땅
- `2` : 목표 지점

목표 지점(`2`)에서 출발하여  
각 `1`까지의 **최단 거리**를 구하는 문제이다.

단,
- 도달할 수 없는 `1`은 `-1` 출력
- `0`은 그대로 `0` 출력

---

## 알고리즘

- BFS (너비 우선 탐색)
- 최단거리 탐색

---

## 풀이 아이디어

이 문제는 **그래프 BFS가 아니라 격자 BFS**이다.

### 핵심 포인트

1. `2`를 시작점으로 BFS 수행
2. 상하좌우 4방향 탐색
3. 이동할 때마다 거리 +1
4. 방문 여부는 `dist` 배열로 관리

---

### BFS 흐름

1. 시작점 `(sy, sx)` 찾기
2. `dist` 배열을 `-1`로 초기화
3. 시작점은 거리 `0`
4. 큐를 이용해 BFS 진행
5. 갈 수 있는 곳(`1`)만 탐색

---

## 주의할 점

- graph[node] 같은 일반 그래프 방식 ❌

    → 반드시 상하좌우 탐색 사용해야 함
- visited 대신 dist == -1로 방문 체크 가능
- 시작점은 거리 0으로 설정
- 도달 못한 1은 자동으로 -1 유지됨

---

## 시간 복잡도

#### BFS: O(N × M)
