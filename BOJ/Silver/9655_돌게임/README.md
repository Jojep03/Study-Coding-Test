# 🧩 BOJ 9655 - 돌 게임

## 📌 문제 설명
두 명의 플레이어 **상근(SK)**과 **창영(CY)**이 번갈아 가며 돌을 가져가는 게임입니다.

- 한 번에 가져갈 수 있는 돌의 개수:
  - 1개 또는 3개
- 마지막 돌을 가져가는 사람이 승리

## 🎯 목표
주어진 돌의 개수 `N`이 주어졌을 때,  
누가 이기는지 출력하는 문제입니다.

---

## 💡 핵심 아이디어

작은 값부터 확인하면 다음과 같은 패턴이 나옵니다:

| N | 결과 |
|--|--|
| 1 | SK |
| 2 | CY |
| 3 | SK |
| 4 | CY |
| 5 | SK |

👉 규칙:
- 홀수 → SK 승리
- 짝수 → CY 승리

---

## 🧠 풀이 방법

- `n % 2 == 1` → `"SK"`
- `n % 2 == 0` → `"CY"`

---

## 시간 복잡도

``O(1)``
