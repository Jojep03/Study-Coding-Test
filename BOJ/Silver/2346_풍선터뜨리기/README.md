# 🧨 BOJ 2346 - 풍선 터뜨리기

## 🔗 문제 링크
https://www.acmicpc.net/problem/2346

---

## 📌 문제 설명

N개의 풍선이 원형으로 놓여 있다.  
각 풍선 안에는 정수가 적혀 있으며, 이를 이용해 다음에 터뜨릴 풍선을 결정한다.

- 첫 번째 풍선을 터뜨린다.
- 풍선 안의 숫자만큼 이동하여 다음 풍선을 터뜨린다.
  - 양수 → 오른쪽 이동
  - 음수 → 왼쪽 이동
- 모든 풍선을 터뜨릴 때까지 반복

👉 터뜨린 풍선의 번호를 순서대로 출력

---

## 💡 핵심 아이디어

### 1️⃣ deque 사용 (회전 구현)

풍선이 원형 구조이므로 `deque`를 사용하면 효율적으로 회전 가능

- 오른쪽 이동 → `popleft()` 후 `append()`
- 왼쪽 이동 → `pop()` 후 `appendleft()`

---

### 2️⃣ (번호, 값) 형태로 저장

풍선 값이 중복될 수 있기 때문에 단순 값으로는 인덱스를 찾을 수 없음

❌ 잘못된 방식
```python
y.index(target)
```

**👉 값이 같으면 항상 첫 번째 인덱스 반환**

#### ✔️ 해결 방법
```(풍선번호, 이동값)```


### 3️⃣ 이동 횟수 주의 (⭐ 중요)

이미 ``popleft()``로 한 칸 이동한 상태이므로

- 양수일 때: move - 1
- 음수일 때: abs(move)

### 🧠 알고리즘 흐름
1. 풍선을 ``(번호, 값)`` 형태로 deque에 저장
2. 하나씩 꺼내면서 결과에 번호 추가
3. 이동값에 따라 ``deque`` 회전
4. 모든 풍선이 사라질 때까지 반복

## ⏱ 시간 복잡도
- 각 풍선마다 최대 N번 회전 가능
- 전체 시간 복잡도: **O(N²)**

**👉 N ≤ 1000이라 충분히 통과**

### 🚨 주의 포인트
- ``index()`` 사용 ❌ (중복 값 문제)
- ``(번호, 값)``으로 관리 ✔️
- ``move - 1`` 처리 꼭 기억
- 마지막 풍선 처리 시 ``break`` 필요
