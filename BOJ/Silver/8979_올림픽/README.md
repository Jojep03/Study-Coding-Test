# 🥇 BOJ 8979 - 올림픽

## 📌 문제 설명
각 나라의 금메달, 은메달, 동메달 수가 주어질 때,  
특정 나라 `K`의 등수를 구하는 문제입니다.

등수는 다음 기준으로 결정됩니다.

1. 금메달 수가 많은 나라가 더 높은 등수
2. 금메달 수가 같으면 은메달 수가 많은 나라가 더 높은 등수
3. 금메달, 은메달 수가 같으면 동메달 수가 많은 나라가 더 높은 등수

---

## 💡 풀이 아이디어

- 먼저 나라 `K`의 메달 수 `(금, 은, 동)`을 저장합니다.
- 모든 나라를 순회하면서 `K`보다 순위가 높은 나라의 개수를 셉니다.
- 최종 등수는 다음과 같습니다.

``등수 = 1 + (나보다 높은 나라 수)``


---

## ⚙️ 핵심 포인트

- 파이썬의 **튜플 비교**를 활용하면 `(금, 은, 동)` 순서대로 자동 비교됩니다.

```python
(gold, silver, bronze) > (t_gold, t_silver, t_bronze)
```

---

## ⏱️ 시간복잡도
- 입력 처리: O(n)
- 순위 비교: O(n)

### 👉 총 O(n)
