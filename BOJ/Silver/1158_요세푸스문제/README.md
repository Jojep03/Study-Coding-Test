# 🥇 BOJ 1158 - 요세푸스 문제

## 🔗 문제 링크

https://www.acmicpc.net/problem/1158

## 📌 문제 설명

N명의 사람이 원형으로 앉아 있고,
K번째 사람을 순서대로 제거합니다.

이 과정을 반복하여 **제거되는 순서**를 출력하는 문제입니다.

출력 형식은 다음과 같습니다.

```
<3, 6, 2, 7, 5, 1, 4>
```


## 💡 풀이 아이디어

이 문제는 **큐(Queue) + 회전(rotate)** 개념을 사용하면 쉽게 해결됩니다.

* 사람들을 `deque`에 넣고 시작
* K번째 사람을 제거하기 위해:

  * 앞에서 K-1명을 뒤로 이동 (`popleft → append`)
  * 그 다음 사람을 제거 (`popleft`)
* 이 과정을 모든 사람이 제거될 때까지 반복

## ⚙️ 알고리즘

1. `deque`에 1부터 N까지 저장
2. 반복:

   * K-1번 회전
   * 맨 앞 사람 제거 → 결과 리스트에 추가
3. 결과를 문제 형식에 맞게 출력

## 🧠 핵심 포인트

* `deque`의 `popleft()`와 `append()`를 활용한 **회전 구현**
* 출력 형식:

  ```python
  "<" + ", ".join(map(str, result)) + ">"
  ```

## ⏱️ 시간 복잡도

* 각 원소가 한 번씩 제거됨 → O(N)
* 회전 포함해도 전체적으로 **O(NK)** 정도지만
  N ≤ 1000이라 충분히 통과
