# 🥋 BOJ 7568 - 덩치

## 📌 문제 설명
사람들의 몸무게와 키가 주어질 때,  
각 사람의 **덩치 등수**를 구하는 문제.

👉 덩치가 크다는 기준:
- **몸무게 AND 키 둘 다 커야 함**

---

## 🧠 풀이 아이디어

각 사람에 대해 모든 사람을 비교하면서:

👉 나보다
- 몸무게도 크고
- 키도 큰 사람이 있다면

→ 내 등수를 1 증가시킨다.

---

## 💡 핵심 공식
**나의 등수 = (나보다 덩치 큰 사람 수) + 1**



---

## ⏱️ 시간복잡도

- 모든 사람을 서로 비교 → **O(N²)**
- N ≤ 50 → 충분히 통과

--- 

## ⚠️ 주의할 점

```python
if weight > target_weight and height > target_height
```

**👉 둘 중 하나만 크면 덩치 큰 것이 아님**
