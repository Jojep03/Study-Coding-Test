# BOJ 1205 등수 구하기

문제 링크  
https://www.acmicpc.net/problem/1205

---

## 문제 설명

랭킹 리스트에 점수들이 **내림차순**으로 주어져 있다.  
새로운 점수 `new`가 주어질 때 이 점수가 **랭킹표에 들어갈 수 있다면 등수를 출력**하고,  
**들어갈 수 없다면 `-1`을 출력**하는 문제이다.

단, 랭킹표에는 **최대 `p`명까지만 등록 가능**하다.

---

## 입력

- `n` : 현재 랭킹에 있는 점수 개수
- `new` : 새로 등록하려는 점수
- `p` : 랭킹표 최대 크기

그 다음 줄에 현재 랭킹 점수들이 **내림차순**으로 주어진다.

---

## 출력

- 새 점수가 랭킹에 들어갈 수 있다면 **등수 출력**
- 들어갈 수 없다면 **-1 출력**

---

## 알고리즘

**구현 (Implementation)**

---

## 풀이 아이디어

### 1️⃣ 랭킹이 비어있는 경우

현재 점수가 하나도 없다면 새 점수는 바로 **1등**이 된다.

```python
if n == 0:
    print(1)
```


### 2️⃣ 랭킹표가 가득 찬 경우

랭킹표가 이미 p명으로 가득 차 있고
새 점수가 마지막 점수보다 작거나 같다면 랭킹에 들어갈 수 없다.

```python
if n == p and score[-1] >= new:
    print(-1)
```

### 3️⃣ 등수 계산

등수는 다음과 같이 계산할 수 있다

```자신보다 큰 점수 개수 + 1```

점수 리스트가 이미 내림차순 정렬되어 있으므로
앞에서부터 new보다 큰 점수만 세면 된다.

```python
cnt = 0
for i in range(n):
    if new < score[i]:
        cnt += 1
    else:
        break
```


### 핵심 포인트

- 랭킹이 가득 찼는지 (n == p) 확인

- 새 점수가 마지막 점수보다 큰지 판단

- 등수는 자신보다 큰 점수 개수 + 1

이 세 가지 조건만 정확히 처리하면 해결할 수 있다.
