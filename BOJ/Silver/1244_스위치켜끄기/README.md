# 🥇 BOJ 1244 - 스위치 켜고 끄기

## 📌 문제 설명
여러 개의 스위치가 있고, 각 스위치는 켜짐(1) 또는 꺼짐(0) 상태를 가진다.  
학생들이 순서대로 스위치를 조작하며, 성별에 따라 방식이 다르다.

- 남학생: 받은 번호의 배수 위치 스위치를 모두 반전
- 여학생: 자신의 번호를 중심으로 좌우 대칭이 같은 구간을 찾아 모두 반전

---

## 💡 풀이 아이디어

### 🔹 남학생
num의 배수 위치를 모두 뒤집는다.

```python
for i in range(num - 1, n, num):
    switch[i] ^= 
```
--- 

## 🔹 여학생

중심을 기준으로 좌우를 비교하며 확장한다.

1. 중심 스위치 먼저 뒤집기
2. 좌우가 같으면 계속 확장
3. 다르면 즉시 중단

```python
switch[x] ^= 1
for i in range(1, y + 1):
    if switch[x - i] == switch[x + i]:
        switch[x - i] ^= 1
        switch[x + i] ^= 1
    else:
        break
```

---
## ⚙️ 핵심 포인트
1. XOR로 토글
```python
switch[i] ^= 1
```

2. 대칭 깨지면 즉시 종료
```python
else:
   break
```

3. 20개씩 출력

```python
if (i + 1) % 20 == 0:
    print()
```
