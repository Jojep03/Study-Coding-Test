# [백준 2217번] 로프 - Python

## 문제 설명

여러 개의 로프가 주어졌을 때, 이 로프들을 이용해서 들어올릴 수 있는 물체의 최대 중량을 구하는 문제이다.

각 로프는 버틸 수 있는 최대 중량이 다르며, 여러 개의 로프를 사용하면 물체의 중량이 각 로프에 고르게 나누어진다.

즉, `k`개의 로프를 사용해서 중량 `W`인 물체를 들어올린다면, 각 로프에는 `W / k`만큼의 중량이 걸린다.

## 문제 풀이 아이디어

로프를 여러 개 사용할 때 중요한 것은 **사용하는 로프 중 가장 약한 로프**이다.

예를 들어 어떤 로프들을 사용했을 때 가장 약한 로프가 `10`까지 버틸 수 있고, 총 `3`개의 로프를 사용한다면 들 수 있는 최대 중량은 다음과 같다.

```text
10 * 3 = 30
```

따라서 로프들을 내림차순으로 정렬한 뒤, 앞에서부터 하나씩 로프를 추가하면서 최대 중량을 계산하면 된다.

## 풀이 과정

1. 로프들을 입력받는다.
2. 로프를 내림차순으로 정렬한다.
3. `i`번째 로프까지 사용한다고 가정한다.
4. 이때 사용할 수 있는 로프 개수는 `i + 1`개이다.
5. 현재 로프가 그중 가장 약한 로프이므로 들 수 있는 중량은 `ropes[i] * (i + 1)`이다.
6. 모든 경우 중 최댓값을 출력한다.

## 코드

```python
import sys

input = sys.stdin.readline

n = int(input())
ropes = [int(input()) for _ in range(n)]

ropes.sort(reverse=True)

res = 0
for i in range(n):
    res = max(res, ropes[i] * (i + 1))

print(res)
```

## 코드 설명

```python
ropes.sort(reverse=True)
```

로프를 버틸 수 있는 중량이 큰 순서대로 정렬한다.

```python
for i in range(n):
    res = max(res, ropes[i] * (i + 1))
```

`i`번째 로프까지 사용한다고 생각한다.

내림차순 정렬을 했기 때문에 `ropes[i]`는 현재 사용하는 로프들 중 가장 약한 로프이다.

따라서 `i + 1`개의 로프를 사용했을 때 들 수 있는 최대 중량은 다음과 같다.

```python
ropes[i] * (i + 1)
```

이 값을 기존 최댓값 `res`와 비교해서 더 큰 값으로 갱신한다.

## 예시

로프가 다음과 같이 주어진 경우:

```text
10
15
20
```

내림차순 정렬하면:

```text
20 15 10
```

각 경우의 최대 중량은 다음과 같다.

```text
20 * 1 = 20
15 * 2 = 30
10 * 3 = 30
```

따라서 정답은 `30`이다.

## 시간 복잡도

로프를 정렬하는 데 `O(N log N)`이 걸리고, 이후 한 번 순회하므로 `O(N)`이 걸린다.

따라서 전체 시간 복잡도는 다음과 같다.

```text
O(N log N)
```

## 정리

이 문제는 로프를 많이 쓸수록 무조건 좋은 것이 아니라, 사용하는 로프 중 가장 약한 로프가 전체 중량을 결정한다.

그래서 로프를 내림차순으로 정렬한 뒤, 현재 로프를 가장 약한 로프로 보고 사용할 수 있는 최대 중량을 계속 갱신하면 된다.
