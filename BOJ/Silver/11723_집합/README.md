# 📘 BOJ 11723 - 집합

## 🔗 문제 정보
- 번호: 11723  
- 난이도: 실버 5  
- 유형: 구현, 자료구조, 비트마스크  

---

## 📌 문제 설명

1부터 20까지의 자연수로 이루어진 집합 S가 있다.  

다음 명령을 처리하는 프로그램을 작성하시오:

| 명령어 | 설명 |
|--------|------|
| add x | S에 x 추가 |
| remove x | S에서 x 제거 |
| check x | x가 있으면 1, 없으면 0 출력 |
| toggle x | 있으면 제거, 없으면 추가 |
| all | S = {1 ~ 20} |
| empty | S = ∅ |

---

## ⚠️ 핵심 포인트

- 최대 연산 수: **3,000,000**
- `in`, `remove` 연산 많음
👉 **자료구조 선택이 핵심**

---

## 🔍 풀이 방법 비교

### ❌ 1. deque 사용 (비추천)

```python
from collections import deque
s = deque()
```

- in → O(N)
- remove → O(N)
- ❌ 시간초과 위험

---

### 2. set 사용

```python
s = set()

s.add(x)
s.discard(x)
x in s
```

- 평균 O(1)
- ✔️ 구현 쉬움
- ✔️ 안정적

---

### 🔥 3. 비트마스크 (최적)


```python
S = 0
```

| 연산 | 코드               | 설명          |      |
| -- |------------------| ----------- | ---- |
| 추가 | `S = (1 << x)`               | x 추가 |
| 제거 | `S &= ~(1 << x)` | x 제거        |      |
| 확인 | `S & (1 << x)`   | 존재 여부       |      |
| 토글 | `S ^= (1 << x)`  | 반전          |      |


### 💡 all 연산 핵심

```python
S = (1 << 21) - 2
```

## ✔️ 의미
- (1 << 21) → 2^21
- (1 << 21) - 1 → 21개 비트 모두 1
- (1 << 21) - 2 → 맨 끝 비트만 0

## ✔️ 결과 (이진수)

`111111111111111111110`

## ✔️ 해석
- bit 1 ~ 20 → 1 → 포함
- bit 0 → 0 → 사용 안 함

👉 즉 {1, 2, ..., 20}

--- 

### ⚡ 시간복잡도 비교 

| 방법      | 시간복잡도 | 추천 |
| ------- | ----- | -- |
| deque   | O(N)  | ❌  |
| set     | O(1)  | ✅  |
| bitmask | O(1)  | 🔥 |
