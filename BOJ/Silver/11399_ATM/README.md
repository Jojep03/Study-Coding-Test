# BOJ 11399 - ATM

## 문제 설명

N명의 사람이 ATM을 이용하려고 한다.

각 사람이 돈을 인출하는 데 걸리는 시간이 주어질 때, 모든 사람이 인출을 완료하기까지 필요한 시간의 합의 최솟값을 구하는 문제이다.

## 아이디어

각 사람이 기다리는 시간의 합을 최소로 만들기 위해서는 인출 시간이 짧은 사람부터 먼저 ATM을 이용해야 한다.

예를 들어 인출 시간이 `3, 1, 4`인 경우,

* 그대로 사용: `(3) + (3+1) + (3+1+4) = 15`
* 정렬 후 사용: `(1) + (1+3) + (1+3+4) = 10`

따라서 인출 시간을 오름차순으로 정렬한 뒤 누적합을 구하면 된다.

## 풀이 과정

1. 인출 시간을 오름차순으로 정렬한다.
2. 현재까지의 누적 시간을 저장한다.
3. 누적 시간을 정답에 계속 더한다.
4. 모든 사람에 대해 반복한 후 결과를 출력한다.

## 코드

```python
import sys

input = sys.stdin.readline

n = int(input())
atms = list(map(int, input().split()))
atms.sort()

x = atms[0]
res = atms[0]

for i in range(1, n):
    x = x + atms[i]
    res += x

print(res)
```

## 코드 설명

```python
atms.sort()
```

인출 시간을 오름차순으로 정렬하여 대기 시간의 총합을 최소화한다.

```python
x = atms[0]
res = atms[0]
```

* `x` : 현재까지의 누적 인출 시간
* `res` : 모든 사람의 대기 시간 합

```python
for i in range(1, n):
    x = x + atms[i]
    res += x
```

현재 사람까지의 누적 시간을 계산하고, 이를 정답에 더한다.

예를 들어 정렬된 배열이

```text
1 2 3 3 4
```

이라면 누적합은

```text
1
3
6
9
13
```

이 되고,

최종 결과는

```text
1 + 3 + 6 + 9 + 13 = 32
```

가 된다.

## 시간복잡도

* 정렬: O(N log N)
* 누적합 계산: O(N)

따라서 전체 시간복잡도는

```text
O(N log N)
```

이다.
