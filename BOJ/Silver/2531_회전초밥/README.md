# [백준 2531] 회전 초밥

## 문제 설명

회전 초밥 벨트 위에 `n`개의 초밥이 놓여 있다.

연속해서 `k`개의 초밥을 먹을 수 있으며, 쿠폰 번호 `c`에 해당하는 초밥을 무료로 하나 더 먹을 수 있다.

먹을 수 있는 초밥 종류의 최댓값을 구하는 문제이다.

---

## 풀이 방법

이 문제는 **슬라이딩 윈도우**와 **원형 배열**을 이용해 해결할 수 있다.

### 1. 처음 `k`개의 초밥 확인

먼저 벨트의 처음부터 `k`개의 초밥을 선택한다.

각 초밥 번호가 현재 구간에 몇 개 있는지 `cnt` 배열에 저장한다.

```python
for i in range(k):
    cnt[lst[i]] += 1
```

---

### 2. 초밥 종류 개수 관리

매번 모든 초밥 종류를 확인하면 시간 초과가 발생할 수 있다.

따라서 현재 먹을 수 있는 초밥 종류의 개수를 `kind` 변수로 관리한다.

새로운 초밥을 추가할 때 기존 개수가 `0`이었다면 새로운 종류가 추가된 것이다.

```python
if cnt[sushi] == 0:
    kind += 1

cnt[sushi] += 1
```

초밥을 제거한 뒤 개수가 `0`이 되었다면 해당 종류가 사라진 것이다.

```python
cnt[sushi] -= 1

if cnt[sushi] == 0:
    kind -= 1
```

---

### 3. 쿠폰 초밥 처리

쿠폰 초밥은 항상 먹을 수 있으므로 처음부터 개수를 `1`로 설정한다.

```python
cnt[c] = 1
kind = 1
```

쿠폰 초밥이 현재 구간에도 존재한다면 개수가 `2` 이상이 된다.

벨트에서 쿠폰 초밥이 빠져도 개수가 `1`로 유지되기 때문에 종류 개수가 감소하지 않는다.

---

### 4. 원형 배열 처리

회전 초밥 벨트는 마지막 초밥 다음에 첫 번째 초밥이 이어진다.

따라서 새로 추가되는 초밥의 위치를 다음과 같이 구한다.

```python
lst[(i + k) % n]
```

인덱스가 `n` 이상이 되면 `% n`을 통해 다시 배열의 처음으로 돌아간다.

---

## 전체 코드

```python
import sys

input = sys.stdin.readline

n, d, k, c = map(int, input().split())
lst = [int(input()) for _ in range(n)]

cnt = [0] * (d + 1)

# 쿠폰 초밥은 항상 포함
cnt[c] = 1
kind = 1

# 처음 k개의 초밥
x = [0] * (d + 1)
x[c] = 1
kind = 1

# 슬라이딩 윈도우

for i in range(k):
    sushi = lst[i]
    if x[sushi] == 0:
        kind += 1
    x[sushi] += 1
res = kind

for i in range(n):
    sushi_x = lst[i]
    x[sushi_x] -= 1
    if x[sushi_x] == 0:
        kind -= 1
    sushi_ad = lst[(i + k) % n]
    if x[sushi_ad] == 0:
        kind += 1
    x[sushi_ad] += 1
    res = max(res, kind)
    
print(res)

```

---

## 시간 복잡도

초밥 벨트를 한 바퀴 돌면서 각 초밥을 한 번씩 추가하고 제거한다.

```text
O(n)
```

초밥 개수를 저장하는 배열 생성까지 포함하면 다음과 같다.

```text
O(n + d)
```

---

## 핵심 정리

* 연속된 `k`개의 구간이므로 슬라이딩 윈도우를 사용한다.
* 회전 벨트는 `% n`을 사용해 원형 배열로 처리한다.
* 현재 초밥 종류 수를 `kind` 변수로 직접 관리한다.
* 쿠폰 초밥을 처음부터 포함하면 별도의 쿠폰 확인 없이 처리할 수 있다.
* 매번 전체 초밥 종류를 검사하면 `O(n × d)`가 되어 시간 초과가 발생할 수 있다.
