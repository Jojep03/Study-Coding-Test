# [백준 1439번] 뒤집기 - Python

## 문제 설명

문자열 `S`는 `0`과 `1`로만 이루어져 있다.

이 문자열에 있는 연속된 숫자들을 한 번에 뒤집을 수 있다.

예를 들어 `0001100`이 있을 때, 가운데 `11`만 뒤집으면 전체 문자열을 `0000000`으로 만들 수 있다.

문자열을 모두 같은 숫자로 만들기 위해 필요한 최소 뒤집기 횟수를 구하는 문제이다.

## 문제 풀이

핵심은 문자열을 **연속된 그룹 단위**로 보는 것이다.

예를 들어,

```text
0001100
```

은 다음과 같이 나눌 수 있다.

```text
000 / 11 / 00
```

여기서 `0` 그룹은 2개, `1` 그룹은 1개이다.

전체를 `0`으로 만들려면 `1` 그룹만 뒤집으면 되고,
전체를 `1`로 만들려면 `0` 그룹 2개를 뒤집어야 한다.

따라서 정답은

```text
min(0 그룹 개수, 1 그룹 개수)
```

가 된다.

## 코드

```python
import sys

input = sys.stdin.readline

s = input().strip()

cnt0 = 0
cnt1 = 0

if s[0] == "0":
    cnt0 += 1
else:
    cnt1 += 1

for i in range(1, len(s)):
    if s[i] != s[i - 1]:
        if s[i] == "0":
            cnt0 += 1
        else:
            cnt1 += 1

print(min(cnt0, cnt1))
```

## 코드 설명


```python
cnt0 = 0
cnt1 = 0
```

`0`으로 이루어진 그룹 개수와 `1`로 이루어진 그룹 개수를 저장할 변수를 만든다.

```python
if s[0] == "0":
    cnt0 += 1
else:
    cnt1 += 1
```

첫 번째 문자는 무조건 하나의 그룹을 시작하므로 먼저 카운트한다.

```python
for i in range(1, len(s)):
    if s[i] != s[i - 1]:
```

앞 문자와 현재 문자가 다르면 새로운 그룹이 시작된 것이다.

```python
if s[i] == "0":
    cnt0 += 1
else:
    cnt1 += 1
```

새로 시작한 그룹이 `0`이면 `cnt0`을 증가시키고,
`1`이면 `cnt1`을 증가시킨다.

```python
print(min(cnt0, cnt1))
```

모든 문자를 같게 만들기 위해서는 더 적은 그룹을 뒤집으면 된다.

따라서 `0` 그룹 개수와 `1` 그룹 개수 중 작은 값을 출력한다.

## 예제

입력:

```text
0001100
```

그룹 나누기:

```text
000 / 11 / 00
```

* `0` 그룹: 2개
* `1` 그룹: 1개

`1` 그룹 하나만 뒤집으면 전체가 `0`이 된다.

출력:

```text
1
```

## 정리

이 문제는 문자열을 하나씩 직접 뒤집는 문제가 아니라,
**연속된 숫자 그룹의 개수를 세는 문제**이다.

`0` 그룹과 `1` 그룹 중 더 적은 쪽을 뒤집으면 최소 횟수가 된다.
